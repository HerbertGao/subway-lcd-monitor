## 新增需求

### 需求:站名特写圆点必须按运行状态着色与闪烁

站名特写场景的圆点必须由列车运行状态驱动着色：当列车处于 `ARRIVING` 状态（站名特写展示正驶向的下一站）时，圆点必须闪烁——在黄色实心与白色实心之间交替、描边恒为深色不变，与线路图下一站圆点的闪烁规律一致；当列车处于 `STOPPED` 或 `DEPARTING` 状态（站名特写展示已停靠的当前站）时，圆点为白色实心、深色描边、不闪烁。圆点的填充色与描边色必须经主题系统的 `ColorConfig` 与 CSS 变量提供。

#### 场景:即将到站时圆点闪烁
- **当** 列车处于 `ARRIVING` 状态、站名特写展示正驶向的下一站
- **那么** 该圆点在黄色实心与白色实心之间闪烁交替、描边恒为深色不变，两态均为实心、不出现空心圆环

#### 场景:停靠或关门时圆点静止白实心
- **当** 列车处于 `STOPPED` 或 `DEPARTING` 状态、站名特写展示已停靠的当前站
- **那么** 该圆点为白色实心、深色描边、不闪烁

## 修改需求

### 需求:线路图站点圆点必须按列车运行状态着色

线路图场景的站点圆点必须由列车运行状态驱动着色：已过站为白色实心；未过站为黄色实心；列车正驶向的下一站（`nextStation`）圆点必须闪烁——在黄色实心与白色实心两态之间交替。下一站圆点的闪烁只改变填充色（黄↔白），其描边必须恒为深色、在闪烁全程保持不变，且闪烁两态均为实心填充、禁止出现空心圆环。圆点的填充色与描边色（含未过站黄色）必须经主题系统的 `ColorConfig` 与 CSS 变量提供。

#### 场景:渲染各状态站点圆点
- **当** 线路图渲染站点圆点
- **那么** 已过站圆点为白色实心、未过站圆点为黄色实心、下一站圆点在黄色实心与白色实心之间闪烁交替，且三类圆点的描边均为深色

#### 场景:下一站闪烁两态均为实心且描边恒定
- **当** 下一站圆点处于闪烁过程中（无论黄色态或白色态）
- **那么** 圆点均为实心填充（黄色或白色）、描边恒为深色不随闪烁改变，不出现空心圆环、不出现描边变色
