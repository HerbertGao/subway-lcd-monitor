# 港铁西铁线 LCD 报站屏 — 界面参考

> 本文档基于参考视频《【港鐵】西鐵綫全程報站及LCD顯示 屯門→紅磡》（1920×1080、576 秒）的均匀采样帧整理，逐一记录港铁车厢 LCD 报站屏每类界面的**构成元素**与**版面位置**，作为 subway-lcd-monitor 后续打磨与期三换乘功能的设计依据。
>
> 代表帧见 `docs/mtr-reference-frames/`。视频本身为设计参考素材、不纳入版本控制（`.gitignore` 中 `*.mkv`）。

---

## 0. LCD 屏总体

- **形态**：超宽横条，宽高比约 7∶2（3.5∶1）。视频画面为 16∶9，LCD 屏居中、上下留黑边。
- **屏底色**：浅灰（约 `#e8e8e8`）。
- **轮播**：列车运行过程中循环切换四类界面 —— 站名特写、近段线路图、全程线路图、全网换乘图。停站/到站期间以站名特写为主，区间运行期间以线路图为主，临近换乘站时插入全网换乘图。

四类界面：

| # | 界面 | 主要出现时机 | 代表帧 |
|---|------|------------|--------|
| 1 | 站名特写 | 停站、关门发车、即将到站 | `01`、`02`、`03` |
| 2 | 近段线路图 | 区间运行 | `04` |
| 3 | 全程线路图 | 区间运行 | `05` |
| 4 | 全网换乘图 | 临近换乘站 | `06` |

## 1. 站名特写界面

代表帧：`01-station-closeup-basic.png`（屯門）、`02-station-closeup-departing.png`（柯士甸）、`03-station-closeup-door-hint.png`（尖東）

### 版面结构（自上而下）

- **上半区（浅灰，约占屏高 75–80%）** —— 站名展示区
  - 水平居中排布一行三件：
    1. **超大中文站名**（黑色粗体）—— 居中行的左段
    2. **站点圆点** —— 居中行的中段，紧贴中文名右侧
    3. **英文站名**（黑色，字号约为中文名的一半）—— 居中行的右段
  - 三者垂直居中于上半区，整体水平居中。
- **下半区（黄色安全条，约占屏高 20–25%）** —— 安全提示
  - 黄条内水平居中：**中文提示语 + 英文提示语**
  - 「請小心空隙」版本的黄条**两端各有一个红圈警示图标**（列车 + 行人，红色圆环禁止标）；图标位于黄条最左与最右。

- **右上角（可选）** —— 开门方向提示
  - 位于上半区右上角，中英双语两行（中文 + 英文小字）。
  - **本侧开门**「請在這邊落車 Please exit this side」：在「绿底圆角胶囊（黑字）」与「黑字无底」两态间**闪烁**，周期约 2s。
  - **对侧开门**「請往另一邊落車 Please exit from the opposite side」：黑字无底、**静态不闪**。
  - 数据无开门方向时该角落留空。

### 文案变体

| 列车状态 | 黄条中文 | 黄条英文 | 黄条两端图标 |
|---------|---------|---------|------------|
| 停站 / 即将到站 | 請小心空隙 | Please mind the gap | **有**（红圈警示图标） |
| 关门发车 | 請勿靠近車門 | Please stand back from the doors | **无** |

> 观察：「請勿靠近車門」版本的黄条略窄、站名行相应略偏下；「請小心空隙」版本黄条略高、带图标。

### 站点圆点

采样帧中同一界面的圆点分别呈白色实心、黄色实心、白色（描边圆环观感）—— 即圆点在**黄↔白之间闪烁**，描边恒为深色。

## 2. 近段线路图界面

代表帧：`04-route-nearby.png`（元朗一带）

### 版面结构

- **整屏为线路图**，浅灰屏底。
- **横向展示当前站附近约 5 个站点**（放大版线路图，字号大于全程图）。
- 每个站点（自上而下）：
  1. **中文站名**（上）—— 已过站灰色、当前站及未过站黑色
  2. **英文站名**（中文名正下方，小字）
  3. **站点圆点** —— 落在线路条上。已过站白色实心、未过站黄色实心、列车正驶向的下一站黄↔白闪烁；均为深色描边。线路图中无单独标记「列车所在站」的特殊圆点。
  4. **换乘标签**（圆点下方，可选）—— 该站可换乘时显示
- **线路条**：水平贯穿所有圆点。**已过段灰色、未过段为线路自身色**（西铁线为品红/紫红）。
- **方向箭头**：位于列车当前所在区间的线路段上（最后一个已过站与下一站圆点之间、约线段中部，不贴圆点），实心三角指向行进方向。**箭头会闪烁** —— 与下一站圆点严格同步，在「绿色实心」与「白色实心（深色描边）」两态间交替（绿色态对应下一站圆点黄色态、白色态对应其白色态）。
- **换乘标签**：换乘站圆点下方画换乘线路的标识 —— 轻铁换乘显示「輕鐵 Light Rail」+ 数个路线编号小框（如 `751` `505` `610`）；其他线路换乘显示该线彩色名牌。

> 观察：采样帧中近段线路图**未见底部蓝色提示条**（与全程线路图不同）。

## 3. 全程线路图界面

代表帧：`05-route-full.png`（嚴禁飲食）

### 版面结构（自上而下）

- **上 / 中部（浅灰，占屏高约 75–80%）** —— 全程线路图
  - **全部站点横向排列**（屯門 … 紅磡，16 站），字号小于近段图。
  - 每站（自上而下）：中文名（上）、英文名（下）、圆点（线路条上）、换乘标签（圆点下方，换乘站才有）。
  - 已过站站名灰、当前站及未过站站名黑。
  - 站点圆点：已过站白色实心、未过站黄色实心、列车正驶向的下一站黄↔白闪烁，均为深色描边。
  - **线路条**水平贯穿：已过段灰、未过段线路自身色（品红/紫红）。
  - **方向箭头**位于列车当前所在区间的线路段上（最后一个已过站与下一站圆点之间），实心三角指向行进方向；**会与下一站圆点同步闪烁**（绿色实心↔白色实心、描边深色）。
  - **换乘标签**：换乘站圆点下方 —— 彩色线路名牌（如「荃灣綫 Tsuen Wan Line」、「東涌綫 Tung Chung Line」、「東鐵綫 East Rail Line」、「高速鐵路 High Speed Rail」含小图标），轻铁换乘为「輕鐵 Light Rail」+ 编号框。
- **底部（深蓝条，占屏高约 15–20%）** —— 提示条
  - 深蓝底、**白色文字、水平居中**。
  - 文案在多条之间轮换，例如：`嚴禁飲食`、`Thank you for travelling on the MTR`、`Change to the … Line at … Station` 等（安全 / 礼仪 / 换乘 / 致谢主题）。

## 4. 全网换乘图界面

代表帧：`06-network-map.png`（东涌线换乘）

### 版面结构（自上而下）

- **上 / 中部（浅灰，占屏高约 80%）** —— 港铁全网络图
  - 显示**整个港铁系统的 2D 网络图**（所有线路交织），而非单线横条。
  - 当前所在线路（西铁线 / 屯马线）以自身色高亮，其余线路淡色显示。
  - 各换乘节点标注线路名牌与中英文站名。
- **右上角**：开门方向提示（同站名特写右上角 —— 本侧开门闪烁绿底↔黑字、对侧开门静态黑字）。
- **底部（深蓝条，占屏高约 15–20%）** —— 换乘提示
  - 深蓝底、白色文字、水平居中。
  - 文案为换乘指引，例如：`往荃灣綫各站，請在此站轉車`、`Tsuen Wan Line interchange`、`Tung Chung Line interchange`、`Light Rail interchange`。

> 该界面在临近换乘站时插入，提示乘客可在此换乘。

## 5. 共通视觉元素

| 元素 | 说明 |
|------|------|
| **站点圆点** | 统一圆形、深色描边。已过站白色实心；未过站黄色实心；列车正驶向的下一站在黄色实心↔白色实心间闪烁、描边恒深色。线路图中**无**单独标记「列车所在站」的特殊圆点 —— 列车位置由方向箭头 + 闪烁的下一站共同表达。换乘站圆点同样落在线路条上。 |
| **方向箭头** | 线路图中位于列车当前所在区间的线路段上（最后一个已过站与下一站圆点之间、约线段中部、不贴圆点），实心三角指向行进方向。**会闪烁** —— 与下一站圆点严格同步，在「绿色实心」与「白色实心（深色描边）」两态间交替。每个线路图界面仅一个方向箭头。 |
| **换乘标签** | 站点圆点下方。彩色线路名牌（线路自身色底/边 + 中英文线路名），部分含小图标（高速铁路）；轻铁换乘为「輕鐵 Light Rail」+ 路线编号小框。 |
| **黄色安全条** | 站名特写下半区。黄底黑字，中英双语；「請小心空隙」版两端带红圈警示图标。 |
| **蓝色提示条** | 线路图 / 全网图底部。深蓝底白字、水平居中，文案在安全/礼仪/换乘/致谢主题间轮换。 |
| **开门方向提示** | 站名特写、全网图的右上角，中英双语两行。**本侧开门**「請在這邊落車」在「绿底圆角胶囊（黑字）」与「黑字无底」间闪烁（周期约 2s）；**对侧开门**「請往另一邊落車」为静态黑字无底、不闪。 |
| **线路条配色** | 已过段灰、未过段线路自身色（西铁线品红/紫红）。 |

## 6. 与当前实现（subway-lcd-monitor）的主要差异

下列差异按「是否在已规划范围」分组，供期三及后续打磨参考。

### 已在期三计划内

- **换乘线路标签缺失** —— 视频中近段图与全程图的换乘站圆点下方均有彩色换乘线路名牌（含轻铁编号框）；当前实现完全不渲染换乘标签。期三 `phase4-mtr-content` 即为此项。

### 尚未规划，建议后续打磨

1. **全网换乘图界面整体缺失** —— 视频有独立的「港铁全网络图 + 换乘提示蓝条」界面（临近换乘站插入）；当前实现只有站名特写 / 近段线路图 / 全程线路图三类场景，无全网图。
2. **站名特写黄条缺少两端警示图标** —— 视频「請小心空隙」黄条两端各有红圈警示图标（列车 + 行人）；当前实现黄条仅有文字。
3. **黄条版式未随文案变化** —— 视频「請勿靠近車門」版黄条略窄、站名行略偏下，「請小心空隙」版略高带图标；当前实现黄条高度/版式固定。
4. **近段线路图底部蓝条** —— 采样帧中视频的近段线路图未见底部蓝条；当前实现近段图带蓝条。需确认是否与视频对齐（移除近段图蓝条）或保留。
5. **全程线路图蓝条文案** —— 视频蓝条文案在安全/礼仪/换乘/致谢多主题间轮换（含 `Change to … Line at … Station`、`Thank you for travelling on the MTR`）；当前实现取自固定提示语集、未含致谢与到站换乘类。
6. **开门方向提示 —— 本侧未闪烁** —— 视频中本侧开门「請在這邊落車」在「绿底圆角胶囊↔黑字无底」间闪烁（周期约 2s），对侧开门「請往另一邊落車」为静态黑字；当前实现本侧（`right`/`both`）为**恒定**绿底胶囊、不闪烁，对侧（`left`）为静态深色字（与视频对侧基本一致）。差异在于本侧应闪烁、当前未闪。
7. **方向箭头位置与闪烁** —— 视频中方向箭头位于列车当前所在区间的线段上（最后一个已过站与下一站之间），且**会与下一站圆点同步闪烁**（绿色实心↔白色实心、描边深色）；当前实现（`useRouteLayout` 的 `directionArrow`）是**静态黑色**三角、紧贴当前站圆点右侧 —— 位置与闪烁均不符。

> 注：线路条颜色由线路数据 `line.color` 驱动（北京线路用各自色），与视频西铁线品红不同属预期，非差异项。
